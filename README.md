# number_guessing_game
This is a simple command-line Number Guessing Game developed using Python.
The program generates a random number between 1 and 50, and the user has 3 attempts to guess the correct number.
After each guess, the program gives feedback whether the guess is too high or too low.
If the user guesses correctly within 3 attempts, they win. Otherwise, the game reveals the correct number.
## Features:
1.Random number generation
2.Limited attempts (3 tries)
3.High/Low feedback
4.Game over message
5.Clean user interaction
